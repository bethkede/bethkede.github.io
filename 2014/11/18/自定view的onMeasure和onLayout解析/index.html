自定view的onMeasure和onLayout解析

onMeasure

onMeasure的作用是告诉父控件你需要的宽度和高度，也就是设置自己本身宽高。

protected void onMeasure(int widthMeasureSpec, int heightMeasureSpec)

首先onMeasure这个方法传递的widthMeasureSpec和heightMeasureSpec这2个参数，这2个参数是父View调用的时候传递过来的。里面包含了View的宽高的类型和数值。

···java
public static int getDefaultSize(int size, int measureSpec) {
int result = size;
int specMode = MeasureSpec.getMode(measureSpec);
int specSize = MeasureSpec.getSize(measureSpec);
switch (specMode) {
case MeasureSpec.UNSPECIFIED:
result = size;
break;
case MeasureSpec.AT_MOST:
case MeasureSpec.EXACTLY:
result = specSize;
break;
}
return result;
}
···
MeasureSpec.EXACTLY：父视图希望子视图的大小应该是specSize中指定的。
MeasureSpec.AT_MOST：子视图的大小最多是specSize中指定的值，也就是说不建议子视图的大小超过specSize中给定的值。
MeasureSpec.UNSPECIFIED：我们可以随意指定视图的大小。

同时MeasureSpec还有静态的方法供我们调用。我们可以讲View的宽高的MeasureSpec修改成我们自己想要的。

public static int makeMeasureSpec (int size, int mode)

如果我们自定义的是一个ViewGroup也需要高度子View我们能给他的宽高和数字和类型。
例如：
···
protected void onMeasure(int widthMeasureSpec, int heightMeasureSpec) {
super.onMeasure(widthMeasureSpec, heightMeasureSpec);
int parentWidth = MeasureSpec.getSize(widthMeasureSpec);
int childWidth = (int) ((parentWidth - 3 driverSize) / 4f);
int count = getChildCount();
for (int i = 0; i < count; i++) {
getChildAt(i).measure(MeasureSpec.makeMeasureSpec(childWidth, MeasureSpec.EXACTLY),
MeasureSpec.makeMeasureSpec(childWidth, MeasureSpec.EXACTLY));
}
int heightCount = count % 4 == 0 ? count / 4 : count / 4 + 1;
int height = heightCount childWidth + (heightCount - 1) * driverSize;
setMeasuredDimension(widthMeasureSpec, MeasureSpec.makeMeasureSpec(height, MeasureSpec.EXACTLY));
}

onLayout

onLayout就是指在ViewGroup里面子View的排布，你想讲第一个子View放在那里。
调用View的layout方法就会拜访这个View的位置。当然如果写的复杂一些会判断Visibility，padding等，写的简单一些也可以。

public void layout (int l, int t, int r, int b)

例如：

···
protected void onLayout(boolean changed, int l, int t, int r, int b) {
int count = getChildCount();
int parentWidth = r - l;
View view;
int left = 0;
int top = 0;
int width = 0;
int height = 0;
for (int i = 0; i < count; i++) {
view = getChildAt(i);
width = view.getMeasuredWidth();
height = view.getMeasuredHeight();

        if (left + width > parentWidth) {
            top += (height + driverSize);
            left = 0;
        }
        view.layout(left, top, left + width, top + height);
        left += (width + driverSize);

    }

}
